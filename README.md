# Unix-Shell-Command-Line-Interpreter-C-
A custom Unix-like shell implemented in C++ that replicates core functionalities of standard shells such as Bash, focusing on process management, job control, and inter-process communication. The shell supports execution of both built-in and external commands using low-level system calls like fork, exec, and waitpid, along with advanced features including background and foreground execution, pipes, I/O redirection, and signal handling for SIGINT, SIGTSTP, and SIGALRM. It maintains an internal jobs list to track running and stopped processes, enabling commands like jobs, fg, and bg, and includes a custom implementation of utilities such as head. 
To run the project, compile it using make smash and then execute the shell with ./smash.
